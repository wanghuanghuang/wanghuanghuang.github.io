Restart.sh 脚本内容：

一．echo命令

Shell的echo指令与PHP的echo指令类似，都是用于字符串的输出。
a) 显示普通字符串
i. echo “It is a test” echo It is a test
这里的双引号完全可以忽略 效果是一样的

ii. 含有转义字符时 若不加引号 则会丢失信息 “”内信息会完整展示

b) 显示转移字符
i. echo “\”It is a test\”” echo \”It is a test\”
这里引号也是可以省略的
